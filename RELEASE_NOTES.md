# Updates

## 0.2.0 (2026-06-20)

✨ 새 기능
- 앱 안에서 버전별 업데이트 내역 보기 (설정 > 업데이트 내역)
- 운동 기록 중 목록에 없는 운동을 검색창에서 바로 생성 (측정 방식 지정, 생성 후 자동 이동)

🛠 개선
- 운동 기록 상세 개편: 부위 태그 + 운동별 최고 기록·이전 대비 표시, 세트 숫자 강조로 가독성 향상
- 운동 화면 스크롤 시 사진이 줄며 제목이 상단에 고정 (collapsing 헤더)
- 최근 기록을 세트 칩으로 간결하게, 최근 3개까지 노출
- 데이터 관리 화면 정리: 초기화 버튼 약화, 복원 안내 문구 완화, 보조 버튼 가시성 개선
- 운동 이미지 공유 레이아웃 개선 (사진 2장 좌우 분할, 최대 2장)
- 흰색 사진 위 슬라이드 점이 안 보이던 문제 수정
